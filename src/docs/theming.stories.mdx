# Theming

You can use Matte's default theme by creating a theme with `createMatteTheme` component,
and then passing it to Material UI's `ThemeProvider`:

```jsx
const theme: MatteTheme = createMatteTheme();

<ThemeProvider theme={theme}>{children}</ThemeProvider>;
```

This is the easiest way to get started with Matte and get the feel
and look you see in this storybook. The default theme includes settings
(colours, texts, shadows and more) for Material UI components and Nivo charts.
You can see it at the end of this page.

## Customising the default theme

You can customise the default theme by passing your own theme.

First create your theme with `createMatteTheme`:

```jsx
// theme.tsx

// Create your theme from the default theme:
// Copy the default theme and change the properties you want
const const myMatteTheme = {
  palette: {
    primary: {
      main: '#727CF5',
    },
    secondary: {
      main: '#7987A1',
    },
    ....
  },
};

// Create the theme
const theme = createMatteTheme(myMatteTheme);

export default theme;
```

Then pass it pass it to Material UI's `ThemeProvider`:

```jsx
// app.tsx

import * as React from 'react';
import * as ReactDOM from 'react-dom';
import { createMuiTheme } from '@material-ui/core/styles';
import { ThemeProvider } from '@material-ui/core/styles';
import theme from './theme';

const app = (
  <MatteThemeProvider theme={theme}>
    <Button />
  </MatteThemeProvider>
);

ReactDOM.render(app, document.getElementById('root'));
```
